Project Overview

Instock is a warehouse and inventory management system developed by a group project team using modern technologies and tools. The project aimed to design and implement a user-friendly system that can help businesses manage their inventory efficiently. The project team worked together to develop a robust and scalable database schema that can handle large amounts of data while ensuring data integrity. The frontend of the application was built using React, which provided an intuitive and seamless user interface. The backend was developed using Node.js and Express.js, which helped to build a fast and reliable API. The system includes features such as real-time inventory tracking, stock level alerts, and automated reordering, which helps businesses to optimize their inventory management processes. The project team used an agile approach with 1-week sprints to ensure the project was delivered on time.

Overall, Instock showcases the project team's ability to collaborate effectively and utilize modern technologies to deliver a high-quality project within a short timeframe.
