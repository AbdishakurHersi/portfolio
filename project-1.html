Project Overview

Proshop is an ecommerce website built using the MERN (MongoDB, Express, React, Node.js) stack. It features an intuitive user interface that allows customers to browse and purchase products. Customers can add products to their cart, create an account, and save their payment and shipping information for future orders. The website uses PayPal for payment processing, which provides a seamless checkout experience for customers. Proshop also includes an admin dashboard that enables site administrators to manage products, orders, and customers. The admin dashboard includes CRUD (Create, Read, Update, and Delete) operations for managing products and orders. Site administrators can view and manage customer information, such as shipping and billing addresses, and view order details and statuses. In addition to the core functionality, Proshop includes features such as user authentication, order tracking, and search functionality to enhance the user experience. The website is responsive, making it accessible on different screen sizes and devices.

Overall, Proshop is a robust ecommerce website that provides a seamless shopping experience for customers and an easy-to-use management system for site administrators.
